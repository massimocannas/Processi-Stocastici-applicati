# Processi-Stocastici-applicati
Questo repository raccoglie materiali, implementazioni ed esempi sviluppati durante il reading course dedicato ai processi stocastici e, in particolare, alla teoria delle catene di Markov in tempo discreto.
Il corso combina aspetti analitici e computazionali, affiancando alla teoria una serie di simulazioni, esperimenti numerici e animazioni (GIF) sui modelli studiati.
